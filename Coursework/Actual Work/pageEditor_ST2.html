Challenges and Lessons Learned

During the implementation of the T-shirt section, several challenges were encountered. One major challenge was ensuring the responsive design of the grid layout. Initially, the items did not align properly on smaller screens, causing overlap and misalignment. This was resolved by using CSS flexbox and grid properties, along with media queries, to adjust the layout dynamically based on screen size. Another challenge was image optimization. High-resolution images slowed down the page load time. To address this, images were compressed without losing quality, and lazy loading was implemented to improve performance.

A third challenge was maintaining consistency in styling across different browsers. Some CSS properties, like gap in flexbox, were not supported in older browsers. This was resolved by adding fallbacks and using polyfills where necessary. Lastly, ensuring accessibility was a priority. Initially, the "Shop this" buttons lacked proper ARIA labels, making them less accessible to screen readers. This was fixed by adding aria-label attributes and ensuring proper contrast ratios for text and background colors.

These challenges taught valuable lessons: the importance of responsive design, the need for performance optimization, and the critical role of accessibility in web development. Overcoming these issues improved the overall user experience and reinforced best practices for future projects.

Compliance

In terms of legal compliance, the site adheres to GDPR by clearly stating how user data is collected and used, and all third-party content (e.g., images and fonts) is properly attributed to avoid copyright violations. Performance optimization techniques, such as image compression and lazy loading, ensure fast load times, while clean, well-documented code supports maintainability. Regular testing and updates are planned to ensure ongoing compliance with Janet regulations, ensuring the site remains secure, accessible, and user-friendly.
